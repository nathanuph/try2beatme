# try2beatme

Web app, users can link their Spotify to battle their friends with their music listening stats

---

## Installation 

1. Clone the repo:
   ```bash
    https://github.com/nathanuph/Spotify-Battle
   ```
    
2. Navigate to the project directory

3. Install dependencies

4. Run the app:
    ```bash
    flask --app myapp run
    ```

### Prerequisites

* Python 3
* Flask
