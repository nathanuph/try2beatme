# try2beatme

Web app, users can link their Spotify to battle their friends with their music listening stats

---

## Installation 

1. Clone the repo:
   ```bash
    https://github.com/nathanuph/Spotify-Battle
   ```
    
2. Navigate to the project directory

3. Install dependencies

4. Run the app:
    ```bash
    python run.py
    ```

    or try

    ```bash
    python3 run.py
    ```

    The website will then be loaded on your local port!

### Prerequisites

* Python 3
* Flask
